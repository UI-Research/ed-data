# Higher education site

## Naming conventions
* College types:
 * pub = public
 * pr = private
 * np = non-profit
 * fp = for-profit
 * 4 = four-year
 * 2 = two-year
* Costs
 * tu = tuition
 * fees = fees
 * rm = room
 * bd = board
* Example: tufeesrmbd_prnp4 = tuition + fees + room + board at four-year private non-profit college
